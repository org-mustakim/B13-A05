1️⃣ What is the difference between var, let, and const?

**Answer:** The difference between var, let, and const is simple. Though <ins>All three are used to declare variable in JavaScript</ins>, but **var is global scoped** and it can be *redeclared or reassignd*, and **let is block-scoped**, it *can't be redeclared* but *can be reassignd*, and **const is also block-scoped* it *can't be redeclared or be reassignd.*





2️⃣ What is the spread operator (...)?

**Answer:** Spread oparator is **a feature added in ES6** it can be used in <ins>iterables to spread it as a single property</ins>. like merging arrays, copying an old array to a new array, or passing arguments.





3️⃣ What is the difference between map(), filter(), and forEach()?

**Answer:** Difference between map(), filter(), and forEach(): 

map() ==> map is used to modify or transform an array. It returns a modified array of the same length as the original one.

filter() ==> filter is used to filter items or elements from an array based on a specific condition 
It returns a new array after filtering the elements from the original array.

forEach ==> forEach is used just to run a specific task on each element of an array. It doesn't return an array; it always returns undefined





4️⃣ What is an arrow function?

**Answer:** Arrow functions are used **to write short function code syntax**.Arrow functions were introduced in **ES6**. While using arrow function, we don't have to write <ins>function</ins> keyword, <ins>curly braces({})</ins> & <ins>return</ins> keyword for small and simple operations. If it contains only one expression, it **automatically returns the result** 





5️⃣ What are template literals?

**Answer:** Template literals or template strings were introduced in **ES6**.Template strings are <ins>defined with backticks</ins>. A dynamic HTML code block can easily be created using template strings.Using template strings *various expression or variables can be embedded easily* like `${expression or variable}`
